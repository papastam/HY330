
GNU Radio out-of-tree module for the CS-330 lesson.

Website: http://www.csd.uoc.gr/~hy330/

Examples and blocks can be found at https://gitlab.com/surligas/gr-tutorial

## Installation

1. `git clone repo`
2. `cd gr-cs330_project2023`
3. `mkdir build`
4. `cd build`
5. `cmake ..`
6. `make`
7. `sudo make install`
